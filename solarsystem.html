Project Overview

Tools Used

Blender 5.1, BlenderKit (textures), Python 3.11

Render Time

N/A

Resolution

1920 x 1080 (1080p)

This animation blends visual storytelling and scientific timing. Each planetary body rotates and revolves with values based on real orbital references while preserving cinematic readability.

Python scripting was used to automate orbital speed control, scene timing, and position accuracy so that each sequence stays consistent during rendering and compositing.

How this was created

Simple pipeline used for each celestial body project:

  1. Create a Python script that generates a spinning planet.
  2. Open Blender and run the script first by going to the Scripting tab. (Make sure to delete everything in the scene first.)
  3. Use BlenderKit (press N in the 3D Viewport and open the BlenderKit menu) to apply your own textures.
  4. Render and enjoy.

Tools used: Blender 5.1, BlenderKit for textures, and Python for scripting.
